글꼴 관련 스타일

font-family 속성 : 텍스트의 글꼴을 지정할 때 사용한다.

선택자 { font-family : 글꼴이름; }의 형식으로 사용된다.

글꼴 변경 테스트 1

** 웹 폰트 사용하는 방법

웹 폰트 제공 사이트 : 구글폰트

font-size 속성 : 텍스트의 크기를 변경할 때 사용한다.

  1. 고정크기 px
  2. 가변크기 em
  3. 가변크기 %
  4. 가변크기 rem

font-weight 속성 : 두꺼운 글씨를 표현할 때 사용한다.

선택자 { fornt-weight : normal | bold | bolder | lighter | 100 | 400 | 900; }의 형식으로 표현
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게
  • 기본글꼴
