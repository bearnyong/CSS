미디어 쿼리

미디어 쿼리는 화면 크기에 따른 각각의 속성 값을 지정하여 여러가지 화면을 구성하는 기술이다.

@media only(생략가능) all(생략가능) (조건문) {실행문}

@media (조건, 조건) {실행문}

  • @media : 미디어 쿼리가 시작됨을 표시한다.
  • only : 미디어 쿼리 구문을 해석하라고 하는 명령어 (생략이 가능)
  • all : 미디어 쿼리를 해석해야 할 대상을 나타낸다. (생략가능)
    • all : 모든 미디어 유형에서 사용할 css를 정의한다.
    • print : 인쇄 장치에서 사용할 css를 정의한다.
    • screen : 컴퓨터 스크린에서 사용할 css를 정의한다.
    • aural : 화면을 읽어 소리로 출력해주는 장치에서 css를 정의한다.
    • tv : TV에서 사용할 css를 정의한다.
    • handheld : 손으로 들고 다니는 장치에서 사용할 css를 정의한다.
    • projection : 프로젝트를 위한 css를 정의한다.
  • and : 앞과 뒤 조건을 나타낸다.(생략가능)
  • (조건) : 해당 조건을 설정할 수 잇음
  • {실행문} : 조건에 따른 실행을 설정한다.
